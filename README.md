# Работа с публичными API на Python

Репозиторий содержит два учебных задания по работе с публичными API с использованием языка **Python**.

Проект демонстрирует:
- отправку HTTP-запросов,
- обработку JSON-ответов,
- работу с внешними API,
- базовую обработку ошибок.

---

## Задание 1 — Получение данных из публичного API

### Описание
В первом задании используется публичный тестовый API **JSONPlaceholder**.

Программа:
- отправляет GET-запрос к эндпоинту `/posts`,
- получает список постов в формате JSON,
- выводит заголовки (`title`) и тексты (`body`) первых 5 постов.

### Используемый API

    https://jsonplaceholder.typicode.com/posts

### Файл

    main.py

### Запуск

    python main.py

---

## Задание 2 — Получение данных о погоде (OpenWeather API)

### Описание
Во втором задании используется API **OpenWeather** для получения текущей погоды.

Программа:
- принимает название города от пользователя,
- отправляет GET-запрос к OpenWeather API,
- получает данные о погоде,
- выводит:
  - температуру,
  - описание погоды,
  - влажность,
  - скорость ветра.

### Используемый API

    https://api.openweathermap.org/data/2.5/weather

Для работы требуется **API-ключ OpenWeather**.

### Файл

    task2.py

### Настройка API-ключа

В файле `task2.py` необходимо указать свой API-ключ:

    API_KEY = "ВАШ_API_КЛЮЧ"

### Запуск

    python task2.py
